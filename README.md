## 安装

```bash
npm install
```

各种依赖参考 [pupa](https://github.com/xwartz/pupa)

## 启用

修改 `_config.yml` 的 `theme` 配置项为 `nuna`:

```yaml
theme: nuna
```

## 更新

``` bash
cd themes/nuna
git pull
```
